#### 项目说明              
* 该项目使用 [Create React App](https://github.com/facebookincubator/create-react-app)脚手架创建的。
* 该项目分别使用了不同的数据状态管理，分别对应不同的分支


#### 如何启动
* start
```
npm start
```
 or
```
yarn start
```
* build
```
npm run build
```
 or
```
yarn build
```

#### 项目框架

该项目使用的react+react-router+mobx+antd全家桶                                                       
欢迎查看[docs](https://www.jianshu.com/p/2d54c2b3cfa3)
